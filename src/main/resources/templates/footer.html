说点什么

没有太多复杂的业务和强大的功能，
就一个简单的网站，把毕业到现在工作的这段时间接触到和学习的一些技术整合到一起，
由一个人独立开发部署；从数据库设计，后台框架搭建，代码编写，linux云服务器部署环境搭建，域名申请和备案，Nginx反向代理、负载均衡、静态资源文件缓存以及Htts的配置；Jenkins自动化部署等
目前所用到的技术: Amaze UI + doT.js + SpringBoot + Mybatis + Mysql + Redis + Nginx + Maven + Jenkins + QinNiu
